# Web Testing – PrestaShop (System Testing)

## Overview
This project focuses on system and end-to-end testing of the PrestaShop e-commerce platform.

The assignment includes designing combinatorial test cases using CTD (Combinatorial Test Design) and implementing automated tests using Cucumber and Selenium.

## Features
- Combinatorial Test Design (CTD)
- Pairwise testing (2-way coverage)
- End-to-end testing scenarios
- Automated UI testing
- Test grouping by execution flow

## Technologies
- Python / Java (depending on your implementation)
- Selenium
- Cucumber (Gherkin)
- PrestaShop (SUT)
- Docker

## Key Concepts
- System testing
- Test design techniques
- Pairwise coverage
- UI automation
- E-commerce workflows

## What is Tested
- Add products to cart
- Apply discounts and coupons
- Shipping and tax calculation
- Checkout flow
- Final price verification

## Skills Demonstrated
- Writing structured test scenarios
- Designing efficient test coverage
- Automating UI tests
- Working with real-world systems
