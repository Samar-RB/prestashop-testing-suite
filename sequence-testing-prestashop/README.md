# Sequence Testing – PrestaShop (Provengo)

## Overview
This project focuses on combinatorial sequence testing of the PrestaShop e-commerce platform.

The assignment includes modeling the checkout flow with Provengo, defining coverage criteria, generating test suites, and validating end-to-end user flows with Selenium.

## Features
- Combinatorial Sequence Testing (CST)
- Behavioral modeling with Provengo
- Coverage-driven test generation
- Two-way coverage criterion
- Domain-specific coverage criterion
- Automated end-to-end test execution

## Technologies
- JavaScript
- Provengo
- Selenium
- PrestaShop
- Docker

## Key Concepts
- Sequence testing
- Behavioral modeling
- Coverage criteria
- Test suite generation
- UI automation

## What is Tested
- Add products to cart
- Checkout flow
- Discounts and pricing behavior
- Shipping and tax scenarios
- Sequence-based user interactions
