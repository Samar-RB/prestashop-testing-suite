# PrestaShop Testing Projects

## Overview
This repository contains two software testing projects focused on the PrestaShop e-commerce platform.

The projects cover end-to-end testing, combinatorial test design, combinatorial sequence testing, and automated test generation for validating pricing, checkout, and user flows.

## Projects

### 1. Web Testing – PrestaShop 
End-to-end system testing of PrestaShop using combinatorial test design, Cucumber, and Selenium.

### 2. Sequence Testing – PrestaShop
Combinatorial sequence testing of PrestaShop using Provengo, coverage criteria, and Selenium-based execution.

## Technologies
- Selenium
- Cucumber / Gherkin
- Provengo
- Docker
- PrestaShop

## Skills Demonstrated
- System testing
- End-to-end automation
- Combinatorial test design
